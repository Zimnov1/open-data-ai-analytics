Тема проєкту: Аналіз доходів територіальних громад Львівської області.

Мета: Дослідити доходи громад Львівської області, визначити громади з найбільшими надходженнями та проаналізувати зміни у часі.

Джерело даних: https://data.gov.ua/dataset/5fe32eab-6d01-4e1a-add5-db5a4ca910fd

Набір даних: "Доходи територіальних громад, всього"

Гіпотези:

1.Львівська громада має найбільші доходи серед громад області.

2.Доходи громад змінюються упродовж кварталів.

3.Існує значна різниця між максимальними та мінімальними доходами громад.
